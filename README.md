## Description

This is a NestJS app with GraphQL

## First Steps

1. Make a copy of the file `.env.template`

2. Rename the new file with `.env.prod`.

3. Now you can edit your `.env.prod` with your custom environments variables.

## Running the app

```bash
$ docker-compose -f docker-compose.prod.yaml --env-file .env.prod up --build
```

- If you don't count with the docker desktop, run this command

```bash
$ docker-compose -f docker-compose.prod.yaml --env-file .env.prod up
```

### To see the available querys and mutations you can visit

- `http://localhost:{{PORT_USED}}/graphql`
